SmartBanking App Redesign

A clean, intuitive, and responsive banking experience designed to improve trust, usability, and user retention for Gen Z and Millennial users

Role: UI/UX Design

Overview

This project aimed to modernize and outdated mobile banking app. The client, a fintech startup, needed a fresh UI and a smoother onboarding experience to reduce user churn and increase trust.

Goals & Objectives

  • Simplify navigation flow
  • Improve account dashboard responsiveness
  • Implement secure but user-friendly login flows

Tools & Tech Stack

Figma | React | Firebase Auth | TailwindCSS

Challenges

A key challenge was simplifying complex financial terms without losing meaning. I worked with a fintech writer to craft user-friendly microcopy and prioritized accessibility from the start, especially for screen readers and keyboard navigation.

Goals & Objectives

Looking to add biometric login and integrate expense tracking analytics in the next version.

Processes

Research & Insights

Conducted 5 user interviews with fintech users aged 22-35 to uncover key pain points and feature gaps.

Ideation & Design

Sketched low-fidelity wireframes, then built high-fidelity mockups in Figma with a clean, mobile-firest layout.

Development

Built responsive UI in React using TailwindCSS and connected fronted to backend with Firebase and REST API.

Results

Ran usability tests with 7 users; 85% completed onboarding unaided. Onboarding time dropped by 40%.
